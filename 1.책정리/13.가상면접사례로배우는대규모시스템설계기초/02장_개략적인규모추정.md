# 개략적인 규모 추정
- 예제)트위터 QPS와 저장소 요구량 추정
    - 가정
        - 월간 능동 사용자는 3억명
        - 50%의 사용자가 매일 사용
        - 각 사용자는 평균적으로 2건의 트윗
        - 미디어 포함 트윗 10%
        - 데이터는 5년간 보관
    - 추정
        - QPS(Query Per Second)
            - 일간 능동 사용자(DAU:Daily Active User) 3억 * 50% = 1.5억
            - QPS = 1.5억 * 2 / 24시간 / 3600초 = 약 3500
            - 최대 QPS = 2 * 3500 = 약 7000
