## 설치
```shell
docker pull grafana/k6 #M1칩 호환성 문제있음
brew install k6
```

## 간단 테스트
- test.js 작성
```
import http from 'k6/http';
import { sleep } from 'k6';

export default function () {
  http.get('https://test.k6.io');
  sleep(1);
}
```
- k6 run
```
k6 run test.js
```
